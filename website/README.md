# chakra-icons showcase website

## Develop

1. Install depedencies on root directory `~/chakra-icons/`

   ```
   yarn | yarn install
   ```

2. Build the package to generate the icons

   ```
   yarn build
   ```

3. Run the project
   in website directory
   ```
   // in website directory
   yarn dev
   // OR
   // in root directory ~/chakra-icons/
   yarn website dev
   ```
